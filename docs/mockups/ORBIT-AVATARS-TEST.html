Проблема: SVG filter размывает ВСЁ внутри — включая фото. Края фото теряют чёткость, мостики между фото = мутная каша из пикселей.

Решение — двойной слой:
1. Нижний слой (в filter): цветные кружки по hue пользователя — дают чистые blob-мостики
2. Верхний слой (ВНЕ filter): фото-аватарки поверх, с clip-path: circle() — чёткие, без размытия

Итог: мостики цветные и чистые, аватарки чёткие. Оба слоя на одном месте.
