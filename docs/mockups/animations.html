blesk animations

Идеи для анимаций и интерактивности. Кликай и наводи.

1. Навигация — Скользящий индикатор

Активный таб подсвечивается зелёным с glow-эффектом. Индикатор плавно скользит между табами. При ховере таб слегка приподнимается.

2. Профиль — Расширяющаяся карточка

Вместо обычного dropdown — пилл плавно расширяется в карточку с меню. Аватар увеличивается, пункты меню появляются каскадно с задержкой. Нажми на пилл.

3. Кнопки — Hover Glow + Ripple

При наведении кнопка подсвечивается изнутри радиальным градиентом. Зелёная кнопка получает glow-тень. Наведи курсор.

4. Список чатов — Каскадное появление

При переключении раздела элементы списка появляются каскадно — каждый с небольшой задержкой, скользя слева. Нажми "Повторить".
