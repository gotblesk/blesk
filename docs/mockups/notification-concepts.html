Notification Concepts
3 уникальных концепта для уведомлений blesk. Выбери какой нравится.
Концепт A
Radial Stack
Карточки вылетают из bell и раскладываются стопкой с лёгким поворотом. Каждая карточка слегка повёрнута — создаёт ощущение "разложенных карт". Friend requests — увеличенные карточки с кнопками. При hover — карточка поднимается и увеличивается. Как колода карт, разложенная веером.

Концепт B
Timeline Ribbon
Вертикальная лента-таймлайн с цветными точками-событиями. Линия таймлайна — градиент от зелёного к синему. Непрочитанные — точки светятся и пульсируют. Прочитанные — пустые точки. Friend request — выделенная карточка с бордером. Каждое событие — точка на линии времени.

Концепт C
Notification Orbs
Уведомления как плавающие сферы вокруг bell — в стиле Nebula. Каждый тип — отдельный orb разного размера и цвета. Friend request — большая сфера с аватаром. Сгруппированные входы — сфера со счётчиком. Orbs плавают и пульсируют. При hover — orb увеличивается и показывает детали. Самый экспериментальный вариант — Nebula для уведомлений.
